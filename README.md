# GreenCare

GreenCare is a full-stack application designed to help users manage and care for their plants.

## Documentation

This repository is split into two main components, each with its own detailed documentation:

- [Frontend Documentation](./frontend/README.md) - React/Vite frontend application
- [Backend Documentation](./backend/README.md) - Backend server application

## Project Structure

```
GreenCare/
├── frontend/    # React + Vite frontend application
└── backend/     # Backend application
```

## Quick Start

For detailed setup and running instructions, please refer to:

- Frontend setup: [Frontend README](./frontend/README.md)
- Backend setup: [Backend README](./backend/README.md)

## Prerequisites

Before you begin, ensure you have the following installed:

- Node.js (v18 or higher)
- npm (for package management)
- Your preferred code editor (VS Code recommended)

## Installation

### Frontend Setup

1. Navigate to the frontend directory:

```bash
cd frontend
```

2. Install dependencies using npm:

```bash
npm install
```

### Backend Setup

1. Navigate to the backend directory:

```bash
cd backend
```

2. Follow backend-specific setup instructions in the backend directory.

## Running the Application

### Frontend Development Server

1. Navigate to the frontend directory:

```bash
cd frontend
```

2. Start the development server:

```bash
npm run dev
```

The application will be available at `http://localhost:8080` by default.

### Building for Production

To create a production build of the frontend:

```bash
cd frontend
npm run build
```

The built files will be available in the `dist` directory.

## Features

- Modern React components using TypeScript
- Responsive design with Tailwind CSS
- Shadcn UI components for consistent UI/UX
- Custom hooks for mobile responsiveness
- Toast notifications system
- Comprehensive UI component library
